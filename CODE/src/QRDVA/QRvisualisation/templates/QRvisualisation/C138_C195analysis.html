Zonal and Quarterly analysis on Maintenance Data

Exploring Maintenance Distribution

A Zonal and Quarterly analysis on Work Order (Maintenance) data

Motivation: While budgeting for maintenance for the upcoming year/quarter, or for figuring out zones which have high maintenance count (and hence are more prone to damage), it is quite useful to divide the work order data into zones (track wise) and quarters (time wise). The data has been divided as such, and some analysis has been carried out on it.

Data Preprocessing

For each of the work order types, the data has been aggregated into zones and quarters. The values represent the number of times the work order of that type has been carried out. in that zone and that quarter.

For this experiment, let us take the case of NUM_ZONES =100. The available data is from the beginning of 2014 to mid of 2019. So, in total we have 22 quarters of data.

Figure 1.Processed Data for work order type = 'Mechanised Resurfacing'

Analysis

1. Ballast Undercutting

Ballast undercutting is one of the most important and costly maintenance works, it involves lifting the tracks to replace the ballast underneath (figure 2)

It seems that the number of work orders for the C138 are far greater than that for C195. Also for C138, it is evident that zone 46-100 undergo considerably heavier maintenance as compared to the other zones. So possibly, these zones suffer from high ballast fouling.
